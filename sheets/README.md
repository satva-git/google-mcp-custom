# Google Sheets MCP Server

An MCP server for Google Sheets integration, providing comprehensive spreadsheet management capabilities.

## Features

- Create, read, and delete spreadsheets with full lifecycle control
- Create and manage individual worksheets (tabs) within spreadsheets
- Update individual cells and ranges with values and formulas
- Apply formula templates with dynamic placeholders across ranges
- Append rows and clear ranges with automatic sheet expansion
- Read spreadsheet data as structured tables or raw cell values

## Usage

This MCP server is designed to run as part of [Obot](https://github.com/obot-platform/obot/). You can run Obot yourself or try it out on our free demo environment at [chat.obot.ai](https://chat.obot.ai).
